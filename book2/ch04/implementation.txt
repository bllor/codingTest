구현 : 머리 속에 있는 알고리즘을 소스코드로 바꾸는 과정

1.구현 시 고려해야할 메모리 제약 사항
파이썬에서 리스트의 데이터 수가 1000만개가 넘을 경우 메모리 사용량이 400mb로 메모리 제한에 걸릴 수 있다.
1000만개 이상 넘어가는 경우는 잘 없겠지만 1000만 개가 넘을 경우 최적화를 고민해봐야 한다.
자신의 코드가 1초에 2000만 번의 연상을 수행한다고 가정하고 문제를 풀면 실행 시간 제한에 안정적이다.
알고리즘 문제를 풀 때는 시간 제한과 데이터의 개수를 먼저 확인한 뒤에 이 문제를 어느 정도 시간 복잡도의 알고리즘으로
작성해야 풀 수 있을 것인지 예측할 수 있어야 한다.

구현의 종류
시뮬레이션유형
- 일련의 명령에 따라서 개체를 차례대로 이동시키는 것
완전탐색유형
- 가능한 경우의 모든 수를 검사해보는 탐색 방법
